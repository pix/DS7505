enum DS7505::FaultTolerance

Fault Tolerance Configuration.

In both operating modes, the user can program the thermostat-fault tolerance, which sets how many con- secutive temperature readings (1, 2, 4, or 6) must fall outside the thermostat limits before the thermostat out- put is triggered. The fault tolerance is set by the F1 and F0 bits in the configuration register. The default factory power-up setting for fault tolerance is 1 (F1 = 0, F0 = 0).
